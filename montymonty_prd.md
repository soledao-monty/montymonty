## 1. 产品概述

### 1.1 产品背景

montymonty（回忆流）是一款专注于记录生活瞬间的应用。用户可以通过简单的操作，记录当下的感受、文字描述、时间、地点以及1-50张照片，形成一个完整的回忆片段。应用采用本地存储方式，保护用户隐私，让用户能够随时回顾珍贵的记忆。

### 1.2 产品定位

- **产品类型**: 移动端应用（HarmonyOS/OpenHarmony）
- **目标用户**: 注重生活记录、希望保存珍贵回忆的用户群体
- **核心价值**: 简单快捷地记录生活瞬间，保留完整的回忆片段

### 1.3 目标设备

- **平台**: HarmonyOS / OpenHarmony
- **设备类型**: 手机 (Phone)
- **SDK版本**: 6.0.2 (API 22)

## 3. 用户界面 (UI) 需求

### 3.1 页面结构

| 页面名称 | 文件路径 | 功能描述 |
|---------|---------|---------|
| 首页 | pages/HomePage.ets | 展示回忆列表，提供创建入口 |
| 创建回忆页 | pages/CreateMemoryPage.ets | 创建新的回忆记录 |
| 回忆详情页 | pages/MemoryDetailPage.ets | 展示回忆详情，支持删除 |

### 3.2 页面详细设计

#### 3.2.1 首页 (HomePage)

- **顶部导航**: 标题"回忆流"，渐变深蓝背景 (#1a1a2e → #16213e → #0f3460)
- **主体内容**: 垂直滚动的回忆卡片列表
- **底部操作**: 浮动"记录此刻"按钮（毛玻璃效果）
- **卡片内容**:
  - 首图缩略图
  - 文字描述（摘要）
  - 时间 + 地点
  - 照片数量标识

#### 3.2.2 创建回忆页 (CreateMemoryPage)

- **顶部导航栏**:
  - 左侧：取消按钮
  - 中间：标题"记录此刻"
  - 右侧：保存按钮
- **文字输入区**: TextArea组件，必填项
- **时间选择区**: DatePicker，默认当前时间
- **地点输入区**: TextInput，可选
- **照片选择区**: MediaGrid组件，3列网格布局

#### 3.2.3 回忆详情页 (MemoryDetailPage)

- **顶部导航**:
  - 左侧：返回按钮
  - 右侧：删除按钮
- **照片展示**: 2列网格布局
- **信息展示区**: 完整文字描述、时间、地点

### 3.3 组件设计

|---------|---------|---------|
| MemoryCard | components/MemoryCard.ets | 回忆卡片展示组件 |

## 4. 数据需求

### 4.1 数据模型

#### 4.1.1 回忆卡片 (MemoryCard)

```typescript
interface MemoryCard {
  id: string;                    // 唯一标识符 (UUID)
  description: string;           // 文字描述 (必填)
  timestamp: number;             // 回忆时间戳 (必填)
  location: string;             // 地点 (可选，默认空字符串)
  mediaUris: string[];          // 照片/视频URI数组 (1-50个)
  createdAt: number;            // 创建时间戳
  updatedAt: number;            // 更新时间戳
}
```

### 4.2 数据存储

| 存储方式 | 存储位置 | 说明 |
|---------|---------|------|
| SQLite | relationalStore | 本地关系型数据库 |
| 数据库名称 | MemoryFlow.db | - |
| 表名 | memory_cards | 回忆卡片表 |

#### 4.2.1 数据库表结构

```sql
CREATE TABLE IF NOT EXISTS memory_cards (
  id TEXT PRIMARY KEY,
  description TEXT NOT NULL,
  timestamp INTEGER NOT NULL,
  location TEXT,
  mediaUris TEXT NOT NULL,
  createdAt INTEGER NOT NULL,
  updatedAt INTEGER NOT NULL
)
```

### 4.3 数据操作接口

| 接口名称 | 功能描述 | 状态 |
|---------|---------|------|
| insert() | 插入新回忆 | 已完成 |
| queryAll() | 查询所有回忆（按时间倒序） | 已完成 |
| queryById() | 根据ID查询单条回忆 | 已完成 |
| update() | 更新回忆 | 已完成 |
| delete() | 删除回忆 | 已完成 |
